# PHP With CodeIgniter
> 생활코드 CodeIgniter

## Environment
|ITEM|VERSION   |
|---|---|
|PHP| 7.4.6  |
|Xdebug| 2.9.6  |
|CodeIgniter| 3.1.11   |

## DB 구동
- Docker를 이용한다.
- 컨테이너 생성 및 실행
```
$ docker-compose up
```
- 컨테이너 시작
```
$ docker-compose start
```
- 컨테이너 중지
```
$ docker-compose stop
```
- 컨테이너 중지 및 삭제
```
$ docker-compose rm
```
